## 📝 Description
This repository contains the first phase of a scalable Social Media Platform backend. Developed with **TypeScript** and **Node.js**, the project emphasizes **Clean Code** principles and a **Modular Architecture** to ensure maintainability and high performance.

---

## 🚀 Key Features
- **User Authentication:** Robust signup and login systems.
- **Environment Support:** Configured for multiple environments (`.env.development`, `.env.production`).
- **Strict Type Safety:** Leveraging TypeScript to minimize runtime errors.
- **Modular Structure:** Distinct separation of concerns (Routes, Controllers, Services).

---

## 🛠️ Tech Stack
- **Runtime:** Node.js
- **Framework:** Express.js
- **Language:** TypeScript
- **Database:** MongoDB (Mongoose ODM)
- **Security:** Bcrypt (Hashing) & JWT (Authentication)
